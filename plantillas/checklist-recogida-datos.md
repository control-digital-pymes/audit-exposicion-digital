# Recogida de datos previa a la auditoría

Información que se pide al cliente antes de empezar. Para una auditoría externa basta con muy poco; el resto es opcional y solo si el cliente quiere ampliar el alcance.

## Imprescindible

- [ ] Dominio o dominios a auditar (principal y secundarios conocidos).
- [ ] Autorización por escrito del titular para realizar el análisis.

## Recomendable (mejora el alcance)

- [ ] ¿Quién gestiona la web actualmente? (interno / agencia / proveedor)
- [ ] ¿Quién gestiona el correo y el DNS?
- [ ] Emails corporativos en uso (para verificación de filtraciones, si se autoriza).
- [ ] ¿Tienen tienda online / procesan pagos / tratan datos personales?
- [ ] Servicios adicionales conocidos (intranet, almacenamiento, paneles).

## Para escalar a análisis con acceso (otro alcance, otra autorización)

- [ ] Acceso a paneles, logs o sistemas (solo con contrato específico).

> Recordatorio: cualquier dato personal del cliente debe anonimizarse antes de procesarse con herramientas de terceros. La transferencia ya es tratamiento a efectos de RGPD.
