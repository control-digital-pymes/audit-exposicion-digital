# Evidencias

En una auditoría real, esta carpeta contiene las salidas de cada comando (whois, dig, headers, testssl, etc.) con sus hashes SHA-256.

En esta versión pública anonimizada **no se incluyen los ficheros de evidencia**, ya que contendrían datos del cliente real. La estructura de la cadena de custodia (nombres de fichero y hashes ilustrativos) puede consultarse en el anexo del informe.
