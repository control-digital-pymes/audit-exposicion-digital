# Glosario para no técnicos

- **DNS:** la "agenda de contactos" de internet; traduce el nombre del dominio a la dirección del servidor.
- **SPF / DKIM / DMARC:** tres mecanismos que, juntos, evitan que alguien envíe correos haciéndose pasar por tu dominio.
- **Cabeceras de seguridad:** instrucciones que tu web da al navegador del visitante para protegerlo de ciertos ataques.
- **Certificado TLS:** lo que hace que tu web use "https" y el candado; cifra la conexión.
- **Certificate Transparency:** registros públicos donde queda anotado cada certificado emitido; permite detectar suplantaciones.
- **Typosquatting:** registrar dominios parecidos al tuyo (con una letra cambiada) para engañar a tus clientes.
- **Subdomain takeover:** cuando un subdominio tuyo apunta a un servicio abandonado que un atacante puede reclamar.
- **Magecart:** robo de datos de tarjeta inyectando código en la página de pago.
- **Cadena de custodia:** registro que demuestra que una evidencia no se ha alterado desde que se recogió.
