# Referencias técnicas

## Estándares y guías
- OWASP Secure Headers Project — cabeceras de seguridad HTTP.
- RFC 7208 (SPF), RFC 6376 (DKIM), RFC 7489 (DMARC).
- NIST SP 800-115 — guía de evaluación de seguridad (referencia metodológica).

## Organismos (España)
- INCIBE — Instituto Nacional de Ciberseguridad. Catálogo de empresas y avisos de seguridad.
- AEPD — Agencia Española de Protección de Datos. Guías de cookies y consentimiento.
- CCN-CERT — referencia del sector público.

## Herramientas
- testssl.sh — https://testssl.sh
- dnstwist — detección de typosquatting.
- crt.sh / Censys — Certificate Transparency.
- Have I Been Pwned — filtraciones conocidas.
- Mozilla Observatory / securityheaders.com — validación de cabeceras.

## Verificación de CAs
- Common CA Database (CCADB) — https://www.ccadb.org
- Mozilla CA Program.
